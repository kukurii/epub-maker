## 💡 技术亮点

### 图片命名系统（完美实现）⭐⭐⭐

**三种场景全覆盖**：
```
1. 手动上传 → img_001.jpg, img_002.png
2. EPUB 导入 → img_001.jpg, img_002.png
3. 批量重命名 → img_001.jpg, img_002.png
```

**工具函数**：
```typescript
// utils/imageNaming.ts
generateImageFilename()      // 生成统一文件名
renameImages()               // 批量重命名
generateNewImageFilename()  // 新上传图片命名
```

**调用位置**：
- ImagesView.tsx - 手动上传 + 批量重命名
- importer.ts - EPUB 导入自动重命名
- useImageUpload.ts - 拖拽上传自动命名
